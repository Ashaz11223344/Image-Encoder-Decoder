How Encoding Works:

  • Your image is converted to pixel data in your browser
  • Pixel values are transformed using a reversible algorithm
  • The result is a noise-like image that hides the original content
  • Noise intensity controls how much the image is scrambled
  • Use the decode function to restore the original image
  • No data is ever sent to any server - everything happens locally
